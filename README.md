## Build Setup:

``` bash
# Download repository:
git clone https://github.com/pyrta2011/toxin.git

# Go to the app:
cd toxin

# Install dependencies:
npm install

# Server with hot reload at http://localhost:8081/
npm run dev

# Output will be at dist/ folder
npm run build
```
